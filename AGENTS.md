## Architecture

```
app.py        Streamlit UI — filters, HTML table rendering, SVG helpers
data.py       yfinance data fetching with st.cache_data; logo resolution
metrics.py    RS Rank (IBD-style quarterly weighted), SMAs, returns, RS vs SPY
tickers.py    Stock universe (STOCKS list), THEME_COLORS, RS_UNIVERSE
pyproject.toml / uv.lock  Dependency declaration and lockfile
```

### Key design decisions

- **Table rendered as single HTML `<table>`** via `st.markdown(unsafe_allow_html=True)` — avoids Streamlit column padding, enables `overflow-x: auto` horizontal scroll, and eliminates `st.components.v1` JS module errors
- **No `st.components.v1.html`** — all charts are inline SVG inside `<td>` cells
- **Filters as flat `st.columns`** — avoids JS conflicts caused by nesting widgets inside `st.expander` + `st.columns`
- **Column widths character-based** — each column sized by worst-case character count; adding a column auto-scales all widths
- **uv over pip/conda** — faster installs, lockfile reproducibility, Python version management

### RS Rank methodology (IBD-style)

```
score = 0.4×Q4 + 0.2×Q3 + 0.2×Q2 + 0.2×Q1   (quarterly weighted 12-month return)
```
Ranked as percentile (1–99) against `RS_UNIVERSE` (curated stocks + large caps + SPY/QQQ/IWM as anchors).

### RS vs SPY (1M) bars

Daily deltas derived from cumulative 20-day RS outperformance vs SPY.
Each bar = 1 trading day. Green = beat SPY, red = lagged. Last bar full-brightness = today.

## Known Quirks & Gotchas

- **yfinance rate limiting**: `fetch_history` batches tickers and is cached with `st.cache_data(ttl=3600)`. Avoid calling outside cache on every rerun.
- **Logo fallback chain**: Yahoo Finance `logo_url` from `info` → Clearbit domain lookup → hidden on error. Small-cap tickers often have no logo.
- **Forward P/E shown as n/a when negative** — negative forward P/E means unprofitable forecast; displaying the raw negative number is misleading.
- **`master` vs `main`**: local repo initialised as `master`. Rename to `main` before pushing to GitHub:
  ```
  git branch -m master main
  ```
